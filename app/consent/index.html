How Adworth works

Every ad has to answer
three questions.

Before any ad is served on Adworth, three things have to be true. If any one fails — the ad doesn’t run. No exceptions. No workarounds.

1
Did the user say yes?
You explicitly grant consent — on your terms, with a time window you choose. No buried checkbox. No pre-ticked opt-in. A real yes that you can revoke any time.
The Seal
2
Is that yes still valid?
Consent expires. Adworth checks that the token is still active, still in scope, and hasn’t been revoked. Stale consent is treated the same as no consent.
The Warden
3
Can we prove it?
Every consent decision is cryptographically signed and written to a permanent ledger. The proof exists whether or not anyone ever asks for it.
The Logbook
